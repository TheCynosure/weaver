# weaver

Weaver weaves a spider tapesty of your favorite bitmap image.

    make; ./weaver path/to/image.bmp threshold

    image:

        any .bmp will do.

    threshold:

        0-255 - determines number of generated triangles.

    Dependencies:

        SDL2

        SDL2_gfx

Hit END to stop the process early. END again to exit.
